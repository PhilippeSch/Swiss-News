# Swiss News Watch App

A native watchOS app that aggregates and displays news from major Swiss news sources directly on your Apple Watch. Stay informed with the latest headlines, sports, culture, and knowledge articles from SRF, NZZ, and 20 Minuten.

## Requirements

- watchOS 11.2 or later
- Xcode 16.2 or later

## Installation

1. Clone the repository
2. Open `Swiss News.xcodeproj` in Xcode
3. Build and run on your Apple Watch or simulator
